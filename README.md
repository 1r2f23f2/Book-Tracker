## Описание программы
**Book Tracker** — это графическое приложение на Python для ведения списка прочитанных книг.  
Программа позволяет:
- Добавлять книги с указанием названия, автора, жанра и количества страниц.
- Просматривать список всех книг в удобной таблице.
- Фильтровать книги по жанру и по минимальному количеству страниц.
- Сохранять и загружать данные в формате JSON (автоматически при добавлении и запуске).
Приложение написано с использованием библиотеки `tkinter` для графического интерфейса.
## Инструкция по запуску
### Требования
- Python 3.6 или выше
- Библиотеки: `tkinter` (входит в стандартную поставку Python)
### Установка и запуск
1. Скачайте файл `book_tracker.py`
2. Откройте терминал/командную строку в папке с файлом
3. Запустите программу командой:
   ```bash
   python book_tracker.py
